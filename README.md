# Simon Says Game
Simon Says is a fun and interactive memory-based game built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat an increasingly long sequence of colors.
<br>

## Features
- Interactive color buttons
- Visual feedback for each move
- Memory-based gameplay
- Increasing difficulty as levels progress
- Displays highest score and current score

## How to Play
1.  Press any key to start the game
2. Watch the sequence of colors shown by the game
3. Repeat the same sequence by clicking the buttons
4. Each level adds a new color to the sequence
5. If you click the wrong color, the game ends

## Tech Stack
- HTML
- CSS
- JavaScript
